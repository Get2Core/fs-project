# 배포 한눈에 보기 👀

## 🎯 가장 빠른 방법: Render (5분)

### 1. GitHub에 올리기
```bash
git add .
git commit -m "Ready for deployment"
git push origin main
```

### 2. Render에서 배포
1. https://render.com 가입 (GitHub 계정)
2. "New +" → "Web Service"
3. 저장소 선택
4. 설정:
   - **Build Command**: `pip install -r requirements.txt && python download_corp_code.py`
   - **Start Command**: `gunicorn app:app`
5. 환경 변수:
   ```
   OPENDART_API_KEY=여기에_키_입력
   GEMINI_API_KEY=여기에_키_입력
   ```
6. "Create Web Service" 클릭
7. 완료! 🎉

## ⚡ 명령어 요약

### 로컬 테스트
```bash
# 의존성 설치
pip install -r requirements.txt

# 회사 데이터 다운로드
python download_corp_code.py

# 서버 실행
python app.py
# 또는
gunicorn app:app
```

### Docker 테스트
```bash
# 이미지 빌드
docker build -t fs-project .

# 컨테이너 실행
docker run -p 5000:5000 \
  -e OPENDART_API_KEY=your_key \
  -e GEMINI_API_KEY=your_key \
  fs-project
```

## 🆘 자주 발생하는 오류

### "Application failed to start"
➡️ `gunicorn`이 requirements.txt에 있는지 확인

### "corp_codes.json not found"
➡️ Build Command에 `python download_corp_code.py` 포함 확인

### "API Key not configured"
➡️ 환경 변수 정확히 입력했는지 확인
