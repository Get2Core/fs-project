# 회사 검색이 안 될 때 빠른 해결법 ⚡

## 🎯 1분 안에 해결하기

### 1단계: 상태 확인 (10초)

브라우저에서 다음 URL 접속:
```
https://your-app-url.com/api/health
```

### 2단계: 결과 판단 (10초)

#### ✅ 정상인 경우
```json
{
  "companies_loaded": 147890  // 큰 숫자 → 정상!
}
```
→ **다른 문제입니다. 브라우저 콘솔 확인**

#### ❌ 문제가 있는 경우
```json
{
  "companies_loaded": 0  // 0 → 문제!
}
```
→ **아래 3단계로 이동**

### 3단계: 환경 변수 확인 (20초)

**Render 사용자:**
1. Dashboard → 서비스 → "Environment" 탭
2. `OPENDART_API_KEY` 값 확인
3. 없거나 잘못됨 → 추가/수정

**Railway 사용자:**
1. Project → "Variables" 탭
2. `OPENDART_API_KEY` 값 확인

### 4단계: 재배포 (30초)

**Render:**
- "Manual Deploy" → "Deploy latest commit"

**Railway:**
- "Deployments" → "Redeploy"

## 🚨 그래도 안 되면

### 긴급 복구: 데이터 재로드

브라우저 개발자 도구 (F12) → Console 탭:

```javascript
fetch('/api/reload-data', { method: 'POST' })
  .then(r => r.json())
  .then(d => console.log(d));
```

성공 메시지:
```json
{
  "success": true,
  "message": "147,890개의 회사 정보를 로드했습니다."
}
```
