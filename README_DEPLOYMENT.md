## 🌟 추천 배포 방법

### Render (무료, 가장 쉬움) ⭐⭐⭐⭐⭐

**장점**:
- ✅ 완전 무료
- ✅ 자동 HTTPS
- ✅ GitHub 자동 배포
- ✅ 설정 간단
- ✅ 초보자 친화적

**배포 시간**: 5-10분

**단계**:
1. GitHub에 코드 푸시
2. Render에서 저장소 연결
3. 환경 변수 설정
4. 배포!

**상세 가이드**: `배포_빠른_시작.md` 참고

---

## 📋 필수 환경 변수

배포 시 반드시 설정:

```bash
OPENDART_API_KEY=your_opendart_api_key
GEMINI_API_KEY=your_gemini_api_key
PYTHONIOENCODING=utf-8
```

**주의**: 따옴표 없이 입력!

## 🔧 Build Commands

### Render/Railway/Fly.io

**Build Command**:
```bash
pip install -r requirements.txt && python download_corp_code.py && python init_db.py
```

**Start Command**:
```bash
gunicorn app:app --bind 0.0.0.0:$PORT --workers 2 --timeout 120
```

```
AI 버튼 클릭 → 완전한 설명 표시 확인
```
