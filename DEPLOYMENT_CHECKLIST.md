## 📋 배포 전 체크리스트

### 1. API 키 준비
- [ ] OpenDart API 키 발급 완료
- [ ] Google Gemini API 키 발급 완료
- [ ] API 키를 안전한 곳에 보관 (비밀번호 관리자 등)

### 2. 코드 준비
- [ ] 모든 변경사항 커밋 완료
- [ ] `.env` 파일이 `.gitignore`에 포함되어 있음
- [ ] `.env.example` 파일이 존재함
- [ ] 민감한 정보가 코드에 하드코딩되지 않았음

### 3. 의존성 확인
- [ ] `requirements.txt`에 모든 필요한 패키지가 포함됨
- [ ] `gunicorn`이 requirements.txt에 포함됨
- [ ] `runtime.txt`에 Python 버전 명시됨

### 4. 배포 파일 확인
- [ ] `Procfile` 존재 및 내용 확인
- [ ] `Dockerfile` 존재 (Docker 배포 시)
- [ ] `.dockerignore` 존재 (Docker 배포 시)

### 5. 로컬 테스트
- [ ] `python app.py`로 로컬 실행 성공
- [ ] 회사 검색 기능 작동

- [ ] 모든 정적 파일 로드 확인 (CSS, JS)

### 6. Git 저장소
- [ ] GitHub 저장소 생성 완료
- [ ] 로컬 코드를 GitHub에 푸시 완료
- [ ] `main` 브랜치가 최신 상태

## 🔍 문제 해결 체크리스트

### "Application failed to start" 오류
- [ ] requirements.txt에 모든 패키지 포함 확인
- [ ] gunicorn 설치 확인
- [ ] Start Command 정확성 확인
- [ ] Python 버전 호환성 확인

### "Module not found" 오류
- [ ] Build Command 실행 확인
- [ ] pip install 성공 확인
- [ ] 가상 환경 활성화 확인 (로컬 테스트 시)

### "corp_codes.json not found" 오류
- [ ] Build Command에 `python download_corp_code.py` 포함 확인
- [ ] OpenDart API 키 정확성 확인
- [ ] 네트워크 연결 확인
- [ ] data/ 디렉토리 생성 확인

### "API Key not configured" 오류
- [ ] 환경 변수 이름 정확성 확인 (대소문자 구분)
- [ ] 환경 변수 값에 공백이나 따옴표 없음 확인
- [ ] 플랫폼에서 환경 변수가 제대로 저장되었는지 확인

### "500 Internal Server Error"
- [ ] 애플리케이션 로그 확인
- [ ] API 키 유효성 확인
- [ ] 외부 API (OpenDart, Gemini) 상태 확인

### 차트가 표시되지 않음
- [ ] 브라우저 콘솔 오류 확인
- [ ] Chart.js CDN 로드 확인
- [ ] 데이터 형식 확인
- [ ] CORS 설정 확인
