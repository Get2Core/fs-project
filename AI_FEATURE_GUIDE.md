# 🤖 AI 재무 분석 기능 사용 가이드

이 문서는 Gemini AI를 활용한 재무제표 설명 기능의 사용 방법을 안내합니다.

## 📌 기능 소개

AI 재무 분석 기능은 Google의 **Gemini AI**를 사용하여 복잡한 재무제표 데이터를 일반인도 이해하기 쉽게 설명해주는 기능입니다.

### 제공되는 분석 내용:

1. **재무 상태 요약**
   - 자산, 부채, 자본의 변화와 의미
   - 재무 구조의 건전성 평가

2. **경영 성과 분석**
   - 매출, 영업이익, 당기순이익의 추세
   - 수익성 분석

3. **주요 특징**
   - 눈에 띄는 변화나 특이사항
   - 전년 대비 주요 지표 변동

4. **투자자 관점**
   - 데이터가 투자자에게 시사하는 점
   - 주요 재무 비율 해석

## 🚀 사용 방법

### 1단계: Gemini API 키 설정

AI 기능을 사용하려면 먼저 Gemini API 키를 설정해야 합니다.

1. [Google AI Studio](https://ai.google.dev/)에 접속하여 API 키 발급
2. 프로젝트 루트의 `.env` 파일에 다음과 같이 추가:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
3. 서버 재시작: `python app.py`

### 2단계: 재무제표 조회

1. 웹 브라우저에서 `http://localhost:5000` 접속
2. 회사명 검색 (예: "삼성전자", "현대차")
3. 사업연도 선택
4. 보고서 유형 선택 (사업보고서 권장)
5. "조회하기" 버튼 클릭

### 3단계: AI 설명 요청

1. 재무제표 차트가 표시되면 상단의 **"🤖 AI 재무 분석"** 섹션 확인
2. **"AI로 쉽게 설명받기"** 버튼 클릭
3. AI가 분석하는 동안 잠시 대기 (약 5-10초)
4. 설명이 표시되면 읽고 재무 상태 이해

## 💡 활용 팁

### 효과적인 사용 방법:

1. **5개년 데이터 조회**
   - AI는 5개년 추세를 분석하므로, 최근 5년간의 데이터가 있는 연도를 선택하세요

2. **연결재무제표 우선**
   - 대기업의 경우 연결재무제표(CFS)를 선택하면 더 포괄적인 분석을 받을 수 있습니다

3. **사업보고서 선택**
   - 연간 사업보고서를 선택하면 가장 완전한 재무 정보를 얻을 수 있습니다

4. **비교 분석**
   - 동일 업종의 다른 회사도 조회하여 AI 설명을 비교해보세요

### 재무제표 구분 선택:

- **연결재무제표 (CFS)**: 대기업, 지주회사, 자회사가 많은 기업
- **개별재무제표 (OFS)**: 중소기업, 단일 사업체

## 📊 AI가 분석하는 주요 재무 비율

AI는 다음과 같은 재무 비율을 자동으로 계산하고 해석합니다:

### 재무 안정성 지표:
- **부채비율** = (부채총계 ÷ 자본총계) × 100
  - 낮을수록 재무 구조가 건전함
  - 일반적으로 200% 이하가 양호

### 수익성 지표:
- **영업이익률** = (영업이익 ÷ 매출액) × 100
  - 본업의 수익성을 나타냄
  - 업종별로 차이가 있음

- **순이익률** = (당기순이익 ÷ 매출액) × 100
  - 최종 수익성을 나타냄
  - 높을수록 효율적인 경영

### 투자 수익률:
- **자기자본이익률(ROE)** = (당기순이익 ÷ 자본총계) × 100
  - 주주 입장의 수익률
  - 높을수록 투자 매력도가 높음

## 🔧 문제 해결

### "Gemini API 키가 설정되지 않았습니다" 오류

**원인**: `.env` 파일에 `GEMINI_API_KEY`가 없거나 잘못됨

**해결**:
1. `.env` 파일 확인
2. API 키가 올바르게 입력되었는지 확인
3. 서버 재시작

### AI 설명이 너무 오래 걸림

**원인**: 네트워크 연결 문제 또는 API 서버 부하

**해결**:
1. 인터넷 연결 확인
2. 잠시 후 다시 시도
3. 브라우저 콘솔(F12)에서 에러 메시지 확인

### "할당량 초과" 오류

**원인**: 무료 할당량(분당 15회, 일일 1,500회) 초과

**해결**:
1. 잠시 기다린 후 다시 시도
2. [Google AI Studio](https://ai.google.dev/)에서 사용량 확인
3. 필요시 유료 플랜 고려

## 📈 AI 설명 예시

### 삼성전자 재무제표 AI 분석 예시:

> **재무 상태 요약:**
> 
> 삼성전자의 자산총계는 2020년 352조원에서 2024년 448조원으로 꾸준히 증가했습니다. 이는 회사가 지속적으로 성장하고 있음을 의미합니다. 부채비율은 약 45%로 매우 낮은 수준을 유지하고 있어, 재무 구조가 매우 건전합니다.
> 
> **경영 성과 분석:**
> 
> 매출액은 5년간 꾸준히 증가하여 2024년 약 300조원을 달성했습니다. 영업이익은 2021년 51조원을 정점으로 반도체 시장 침체로 인해 2023년에 일시적으로 감소했으나, 2024년 회복세를 보이고 있습니다. 영업이익률은 약 12%로 양호한 수준입니다.
> 
> **주요 특징:**
> 
> 2023년 반도체 업황 악화로 영업이익이 크게 감소했으나, 2024년 AI 반도체 수요 증가로 빠르게 회복했습니다. 순이익률은 약 9%로, 글로벌 IT 기업 중 상위권 수준입니다.
> 
> **투자자 관점:**
> 
> 자기자본이익률(ROE)이 약 10%로 안정적이며, 낮은 부채비율은 향후 추가 투자 여력이 충분함을 시사합니다. AI 반도체 시장의 성장과 함께 실적 개선이 기대됩니다.
